## âš™ï¸ Note about licensing and contribution

The licensing and citation documents (`LICENSE.md`, `CITATION.cff`)  
are managed at the **root level of the Docs as System repository**  
and therefore are not part of this template.  

Contribution and review processes are managed through the agent operational policy (`AGENT_OPERATIONAL_POLICY.md`)  
and not through a separate `CONTRIBUTING.md` file.

> There is no need for licensing or contribution files inside the `project` folder.  
> The methodology itself already defines the rules at the system level.
